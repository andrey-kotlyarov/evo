Сам интерпретатор можно скачать тут
http://www.warmplace.ru/soft/pixilang/index_ru.php

Запускаете файл "pixilang.exe" и в диалоговом окне указываете
файл с кодом.("evolution", "viewer")

Файл "evolution" - запуск моделирования. 
Скрипт проводит 4 моделирования подряд 
и сохраняет результаты в файлах graf01.g, graf02.g, graf03.g, graf04.g
Можно выключить в любой момент, но сохранятся только те результаты моделирования,
которые доведенны до конца.
При клике по полю меняется режим просмотра. Либо отображается каждый шаг, либо только последний шаг в поколении. Скорость работы при этом возрастает наверно в десятки раз.


настройки мира в файле "constant"

Файл "viewer" - просмотор результата моделирования в виде графика
Имя файла с данными прописанно в коде "viewer" в виде переменной file2view

Для просмотра и редактирования кода использую редактор Notepad++.
Файлы должны быть в кодировке UTF-8

Разрешение моего планшета 800*480 пикселей. 
Под это разрешение и сделан проект.
